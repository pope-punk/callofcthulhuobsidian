## Description

**Polybius** refers to the mind-altering arcade game created by Andrew Chamberlain (later [[Anselm Chambers (Deacon)|Deacon Anselm Chambers]]) using principles derived from [[The Foxglove Algorithm]][cite: 583]. It formed the basis for his cult activities in Seattle.

## Known Information & Connections

* **Creator**: Andrew Chamberlain.
* **Basis**: Used [[The Foxglove Algorithm]] principles].
* **Cult Connection**: Associated with Chamberlain's cult, which was raided by BSSA in 1984[cite: 585].
* **[[Operation Colossus]] Link**: Chamberlain's experience with Polybius influenced BSSA's approach in Operation Colossus, particularly the person-machine feedback loops using audio patterns.
