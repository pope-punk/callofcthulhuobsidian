## Overview
The hidden headquarters of the [[BNJC (Blue Note Jazz Collective)]], located beneath their upscale jazz club cover identity in downtown [[Sequoia Falls, WA]].

## Access
* Via a coded elevator within the public jazz club. Requires code **13762**.

## Features
* **Underground Facility:** Sophisticated base for [[Mirror-Breaker Protocol|Mirror-Breaker]] operations.
* **Security:** Likely employs jazz-based metaphors or potentially sonic/pattern-based defenses aligned with [[BNJC (Blue Note Jazz Collective)|BNJC]] philosophy.
* **Technology:** Houses "impulse-class radiometry" equipment for detecting anomalous signals/patterns.
* **Library:** Contains archives potentially including [[IHRU (InfoHazard Response Unit)|IHRU]] records, [[CALLISTO Report]], files on [[EUROPA (1986-1992)]], [[Alice Redmond (Agent Altissimo)]], [[Dylan Hill (Agent Destination)]], and potentially other [[Mirror-Breaker Protocol|Mirror-Breaker network]].
* **Personnel:** Hub for [[BNJC (Blue Note Jazz Collective)|BNJC]] agents including [[Nigel Christmas (Nine-String)]], [[Travis Ward (Agent Tremolo)]], [[Agent Chrysalis]], [[Leah Burns (Agent Legato)]], and the mole [[Steven Rober (Agent Shutter)]].

## Significance
* Primary point of contact and potential support base for players allied with [[BNJC (Blue Note Jazz Collective)|BNJC]].
* Key source of lore, intelligence, and potentially specialized equipment.
* Site where critical intelligence ([[Montreal Phone Number]]) was compromised by [[Steven Rober (Agent Shutter)]].
